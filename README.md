# Loan Approval Prediction App

An AI-powered Machine Learning application that predicts whether a loan will be approved based on applicant financial and personal details.

## Live Demo
🔗 https://loan-approval-app-mqkakjklcw4mgbpu3ursec.streamlit.app/

---

## Project Overview
This project uses a trained ML model to analyze key applicant features like income, credit score, loan amount, and financial history to determine loan approval status.

It provides an interactive UI where users can input their details and get instant predictions.

---

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

---

## Machine Learning Model
- Model Used: Random Forest Classifier
- Preprocessing:
  - Handling missing values
  - Encoding categorical features
  - Feature scaling
- Output:
  - Loan Approved / Rejected
  - Prediction Confidence Score

---

## Features
- Interactive web UI
- Real-time prediction
- Confidence score display
- Handles multiple financial parameters
- Fully deployed on cloud
